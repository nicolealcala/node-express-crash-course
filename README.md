# Beginner Node.js Web Server

<br/>

This project is created as part of **The Odin Project** Node.js curriculum. It serves as my _first Node.js project_, where I build a simple web server using only **core Node.js modules** (no Express). The Server responds with basic HTML pages and demonstrates how a minimal backend works under the hood.
